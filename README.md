# ClarySwap - Stacks AMM MVP scaffold

This repo contains:
- `contracts/` : Clarity contracts (factory, pair, router)
- `frontend/` : Next.js frontend skeleton (pnpm)
- `Clarinet.toml` : Clarinet config

Local quickstart:
1) Install pnpm, Node 18+, Rust+cargo (for Clarinet)
2) Frontend:
   cd frontend
   pnpm install
   pnpm dev
3) Clarinet:
   cargo install --locked clarinet
   clarinet test
